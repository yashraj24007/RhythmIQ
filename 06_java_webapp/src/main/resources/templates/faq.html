General Questions

What is RhythmIQ? +

RhythmIQ is an AI-powered ECG analysis platform that uses machine learning to classify cardiac rhythms from ECG images. It can identify six types of rhythms: Normal (N), Supraventricular (S), Ventricular (V), Fusion (F), Unknown (Q), and Paced (M) beats.

Is RhythmIQ free to use? +

Yes, RhythmIQ is currently free for educational and research purposes. We offer unlimited ECG analyses with no subscription required.

Can I use RhythmIQ for clinical diagnosis? +

No. RhythmIQ is designed exclusively for educational and research purposes. It should NOT be used for clinical diagnosis, treatment decisions, or patient care. Always consult qualified healthcare professionals for medical advice.

Using RhythmIQ

What file formats are supported? +

We support the following image formats:

  • JPEG (.jpg, .jpeg)
  • PNG (.png)

Maximum file size: 10MB per image

How long does analysis take? +

Most ECG analyses complete in under 2 seconds. Processing time may vary based on image size, network speed, and server load.

What image quality do I need? +

For best results:

  • Use clear, high-resolution images (256x256 pixels minimum)
  • Ensure ECG waveforms are clearly visible
  • Avoid blurry or low-quality images
  • Remove any patient identifying information
How accurate is RhythmIQ? +

Our AI model achieves approximately 98.3% accuracy on test datasets. However, accuracy can vary based on image quality and complexity. Always verify results with professional medical evaluation.

Technical Issues

Why is my upload failing? +

Common reasons for upload failures:

  • File size exceeds 10MB limit
  • Unsupported file format
  • Poor internet connection
  • Browser compatibility issues

Try reducing file size, checking your connection, or using a different browser.

Which browsers are supported? +

RhythmIQ works best on modern browsers:

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

Please ensure your browser is up to date.

Why are results taking longer than expected? +

Slow analysis can be caused by:

  • Large image file sizes
  • Slow internet connection
  • High server load during peak hours
  • Browser performance issues

Try compressing images, checking your connection speed, or analyzing during off-peak hours.

Privacy & Security

Is my data secure? +

Yes. We implement industry-standard security measures:

  • All connections use SSL/TLS encryption
  • Images are processed securely
  • No permanent storage of uploaded images
  • Compliance with data protection regulations
Do you store my ECG images? +

No. ECG images are processed in real-time and are not permanently stored on our servers. Images are automatically deleted after analysis is complete.

Is RhythmIQ HIPAA compliant? +

RhythmIQ is designed for educational purposes and is not HIPAA-certified for clinical use. Do not upload ECG images containing protected health information (PHI) or patient-identifying data.
